Ce projet est une application web de gestion de tâches (To-Do List) développée en HTML, CSS et JavaScript. Elle permet aux utilisateurs d’organiser efficacement leurs activités quotidiennes en leur offrant la possibilité d’ajouter, de cocher ou de supprimer des tâches.

L’application utilise le stockage local du navigateur (localStorage) pour conserver les données même après la fermeture de la page, garantissant ainsi une expérience utilisateur fluide et continue.
